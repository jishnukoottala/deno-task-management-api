## REST API using Deno - BOOKS API - (CRUD)

- Install denon
  `deno install --allow-read --allow-run --allow-write -f --unstable https://deno.land/x/denon/denon.ts`

- to generate a script file
  `denon init`

- Place your deno script in denon.json file

- To start
  `denon start`
